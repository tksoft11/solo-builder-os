# Example Project: Thai Calendar PWA

This directory contains a complete, end-to-end example of a product built using the **Solo Builder OS** workflow. 

## The Original Idea
> "I want to build a simple web app that shows Thai public holidays. The official government sites are hard to read on mobile, and people always forget when the next long weekend is."

## How this was generated
The creator of this example took the raw idea above and ran it through the Solo Builder OS Agents in sequence. The resulting files in this directory are the raw, unedited outputs from those agents.

## Files
1. **[01-strategy.md](01-strategy.md)** - The foundational MVP scope and JTBD defined by the *Product Strategist*.
2. **[02-market-research.md](02-market-research.md)** - Competitive analysis and Moat defined by the *Market Researcher*.
3. **[03-prd.md](03-prd.md)** - The rigorous requirement document created by the *PRD Writer*.
4. **[04-ux-and-code.md](04-ux-and-code.md)** - The interface flow and technical architecture defined by the *UX Planner* and *Code Planner*.
5. **[05-growth-and-launch.md](05-growth-and-launch.md)** - Marketing copy, pricing, and launch steps from the *Growth Marketer*, *Monetization Advisor*, and *Launch Operator*.
