# Solo Builder OS - Agents

This document provides an index and overview of all the AI Agents available in the **Solo Builder OS**. Each agent is designed with a specific role, inputs, and outputs to help solo founders execute their projects efficiently.

## Core Agents (World-Class Edition)

| Agent Name | Role & Purpose | Status |
| :--- | :--- | :--- |
| **[Product Strategist](agents/product-strategist.md)** | Helps turn vague ideas into a clear, validated product concept using First Principles and JTBD. | ✅ Available |
| **[Market Researcher](agents/market-researcher.md)** | Analyzes the competitive landscape, positioning gaps, and Blue Ocean opportunities. | ✅ Available |
| **[PRD Writer](agents/prd-writer.md)** | Synthesizes strategy into a MECE-structured Product Requirements Document (PRD). | ✅ Available |
| **[Code Planner](agents/code-planner.md)** | Translates the PRD into build-ready architecture (SOLID) and issue-ready tasks. | ✅ Available |
| **[UX Planner](agents/ux-planner.md)** | Defines frictionless user flows and Information Architecture using Behavioral Economics. | ✅ Available |
| **[Growth Marketer](agents/growth-marketer.md)** | Creates high-converting landing page copy and launch messages using Direct-Response frameworks. | ✅ Available |
| **[Monetization Advisor](agents/monetization-advisor.md)** | Proposes Value-based pricing models and revenue optimization strategies. | ✅ Available |
| **[Launch Operator](agents/launch-operator.md)** | Orchestrates the launch campaign, distribution channels, and tracks early traction KPIs. | ✅ Available |
| **[Quality Evaluator](agents/quality-evaluator.md)** | Acts as a Red Team to critique outputs, enforce quality, and maintain the Feedback Loop. | ✅ Available |

## How to use the Agents
1. Navigate to the `agents/` directory.
2. Open the Markdown file for the agent you need.
3. Copy the prompt/system instructions provided in the file.
4. Paste it into your preferred AI tool (e.g., ChatGPT, Claude) to initialize the agent.
5. Provide the required **Inputs** as listed in the agent's file.
